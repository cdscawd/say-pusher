interface
ispring.presenter.player.restriction.INavigationRestrictionReason
The INavigationRestrictionReason interface describes the reason for presentation navigation restriction.

To obtain the INavigationRestrictionReason interface use the restrictionReason() method on the INavigationRestriction interface.

Methods

relatedSlideIndex() → {number|undefined}

Returns the slide index related to the navigation restriction reason type or undefined if the restriction reason type does not assume the related slide.
Returns:
Type: number | undefined

type() → {string}

Returns the navigation restriction type.

The ispring.presenter.player.restriction.NavigationRestrictionReasonType object properties define the possible values of the type() method.
