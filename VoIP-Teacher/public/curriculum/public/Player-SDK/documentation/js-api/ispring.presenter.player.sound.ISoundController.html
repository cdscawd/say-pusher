interface
ispring.presenter.player.sound.ISoundController
The ISoundController interface is used to control the sound volume level of the presentation.

To obtain the ISoundController interface from the presentation view use the soundController() method on the IPresentationView interface.

See also
  • ispring.presenter.player.IPresentationView#soundController

realVolume() → {number}

Returns the real sound volume level takin the muted state into account.

If the sound is muted, the realVolume() method returns 0. Otherwise the value of the [volume()]volume() method is returned.

volume() → {number}

Returns the sound volume, ranging from 0 (silence) to 1 (full volume).

The real sound volume level depends on the muted state of the sound controller. To obtain the real sound volume level use the realVolume() method.
