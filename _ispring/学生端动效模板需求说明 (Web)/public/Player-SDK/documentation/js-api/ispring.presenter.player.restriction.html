(static) NavigationActionType :string

The NavigationActionType object properties provide values for the navigationAction() method of the INavigationRestriction interface.
Type: string
Properties:
Name Type Default Description
SWITCH_TO_NEXT_SLIDE string switchToNextSlide Corresponds to the gotoNextSlide() method call on the IPresentationPlaybackController interface.
SWITCH_TO_PREVIOUS_SLIDE string switchToPreviousSlide Corresponds to the gotoPreviousSlide() method call on the IPresentationPlaybackController interface.
ARBITRARY_SLIDE_SWITCHING string arbitrarySlideSwitching Corresponds to the gotoSlide() method call on the IPresentationPlaybackController interface.
SLIDE_SHOW_CONTROL string slideShowControl Reserved.
SWITCH_TO_NEXT_STEP string switchToNextStep Corresponds to the gotoNextStep() method call on the IPresentationPlaybackController interface.
SWITCH_TO_PREVIOUS_STEP string switchToPreviousStep Corresponds to the gotoPreviousStep() method call on the IPresentationPlaybackController interface.
PLAY_PAUSE_CONTROL string playPauseControl Corresponds to the play() and pause() method calls on the IPresentationPlaybackController interface.
PRESENTATION_SEEKING string presentationSeeking Corresponds to the gotoTimestamp() method call on the IPresentationPlaybackController interface with the target timestamp is outside the current slide.
SLIDE_SEEKING string slideSeeking Corresponds to the gotoTimestamp() method call on the IPresentationPlaybackController interface when the target timestamp is within the current slide.
QUIZ_SWITCH_TO_NEXT_SLIDE string quizSwitchToNextSlide Corresponds to the gotoNextSlide() method call on the IPresentationPlaybackController interface when the current slide is a quiz and branching rules are applied to the current slide.
QUIZ_SWITCH_TO_NEXT_SLIDE_WITHOUT_BRANCHING string quizSwitchToNextSlideWithoutBranching Corresponds to the gotoNextSlide() method call on the IPresentationPlaybackController interface when the current slide is a quiz and no branching rules are applied to the current slide.
QUIZ_ARBITRARY_SLIDE_SWITCHING string quizArbitrarySlideSwitching Corresponds to the gotoSlide() method call on the IPresentationPlaybackController interface when the current slide is a quiz.

(static) NavigationRestrictionReasonType :string

The NavigationRestrictionReasonType object properties provide values for the type() method of the INavigationRestrictionReason interface.
Type: string
Properties:
Name Type Default Description
CURRENT_SLIDE_IS_LOCKED string currentSlideIsLocked Indicates that navigation is locked on the current slide.
CURRENT_SLIDE_IS_NOT_COMPLETED string currentSlideIsNotCompleted Indicates that the user must complete the current slide before leaving it.
CURRENT_SLIDE_IS_FIRST_SLIDE string currentSlideIsFirstSlide Indicates that it is not allowed to move to the previous slide because the current slide is the first one.
CURRENT_SLIDE_IS_LAST_SLIDE string currentSlideIsLastSlide Indicates that it is not allowed to move to the next slide because the current slide is the last one.
CURRENT_SLIDE_IS_INTERACTION string currentSlideIsInteraction Indicates that the play/pause actions are not available because the current slide is interactive, e.g. contains a quiz or an interaction.
BACKWARD_NAVIGATION_IS_RESTRICTED string backwardNavigationIsRestricted Indicates that backward navigation is not allowed.
FORWARD_NAVIGATION_IS_RESTRICTED string forwardNavigationIsRestricted Indicates that forward navigation is not allowed.
PRESENTATION_SEEKING_DISABLED string presentationSeekingDisabled Indicates that the seek operation is not allowed.
INTERACTION_NOT_COMPLETED string interactionNotCompleted Indicates that the user is not allowed to leave the current slide until completing the quiz or interactive object.
PRECEDING_QUIZ_NOT_PASSED string precedingQuizNotPassed Indicates that moving to the slide is not allowed until passing the quiz on one of preceeding slides.
PRECEDING_QUIZ_NOT_COMPLETED string precedingQuizNotCompleted Indicates that moving to the slide is not allowed until completing the quiz on one of preceeding slides.
PRECEDING_QUIZ_FAILED string precedingQuizFailed Indicates that moving to the slide is not allowed because the user failed the quiz on one of the preceeding slides.
