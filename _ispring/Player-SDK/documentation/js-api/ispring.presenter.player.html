Members

(static) PresentationPlaybackState :string

The PresentationPlaybackState object properties provide values for the playbackState() method of the IPresentationPlaybackController interface.
Type: string
Properties:
Name Type Default Description
PLAYING_SLIDE string playingSlide Indicates that the presentation playback controller is playing a slide.
PLAYING_TRANSITION string playingTransition Indicates that the presentation playback controller is playing a slide transition effect.
PAUSED_TRANSITION string pausedTransition Indicates that the presentation playback controller paused a slide transition.
PAUSED_SLIDE string pausedSlide Indicates that the presentation playback controller paused a slide.
SUSPENDED string suspended Indicates that the presentation playback controller paused a slide transition.
BUFFERING string buffering Indicates that the presentation playback controller is buffering data.
