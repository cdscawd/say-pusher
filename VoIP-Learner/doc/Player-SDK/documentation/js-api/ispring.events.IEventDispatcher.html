interface
ispring.events.IEventDispatcher
The IEventDispatcher interface provides methods for adding or removing event handlers.

Methods

addHandler(fn non-null, context opt, priority opt)

Registers an event handling function to be invoked when the event dispatcher dispatches the event.
Parameters:
Name Type Attributes Description
fn function The event handling function to be invoked when the event is dispatched.
context Object <optional>
 The invocation context of the event handling function to be used as this parameter.
priority number <optional>
 The event handler priority, zero by default.

removeHandler(fn non-null, context opt, priority opt)

Removes an event handler from the event dispatcher object.
Parameters:
Name Type Attributes Description
fn function The event handling function to rename.
context Object <optional>
 The invocation context of the event handling function.
priority number <optional>
 The event handler priority, zero by default.
