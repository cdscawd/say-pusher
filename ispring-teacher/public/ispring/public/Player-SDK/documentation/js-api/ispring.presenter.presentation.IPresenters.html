interface
ispring.presenter.presentation.IPresenters
The IPresenters interface represents a collection of the presentation presenters.

The IPresenters interface of the loaded presentation can be obtained using the presenters() method on the IPresentation interface.

See also
  • ispring.presenter.presentation.IPresenter
