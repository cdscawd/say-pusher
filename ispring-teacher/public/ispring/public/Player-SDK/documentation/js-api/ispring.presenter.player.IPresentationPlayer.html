interface
ispring.presenter.player.IPresentationPlayer
The IPresentationPlayer interface provides access to the presentation and its view.

To obtain the IPresentationPlayer interface of the presentation, declare the ispringPresentationConnector object in the global scope (on the window object) and load the presentation html using <iframe> element. The iSpring-generated presentation will call the register() method on the ispringPresentationConnector object and pass the IPresentation player interface as a parameter.

The following HTML code illustrates how to load the presentation and obtain its IPresentationPlayer interface.

<html>
<body>
<iframe id="frame" src="presentation.html" style="width:800px;height:600px;"></iframe>
<script>
var ispringPresentationConnector = {}
ispringPresentationConnector.register = function(player)
{
    log("Player initialized");
);
</script>
</body>
</html>
