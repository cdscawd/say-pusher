interface
ispring.presenter.presentation.resources.references.IReference
The IReference interface provides the presentation reference information.

Use the getReference() method on the IReferences interface to access the presenation references.

Methods

navigate(target opt)

Navigates the reference in the specified target window.
Parameters:
Name Type Attributes Description
target string <optional>
 The target window name to navigate the reference in. If this parameter is omitted, the reference is opened in the window, returned by the target() method.
