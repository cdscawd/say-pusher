interface
ispring.presenter.player.IPresentationView
The IPresentationView interface represents the presentation view responsible for displaying presentation content

The IPresentationView methods also provide access to the sound controller and playback controllers.

The IPresentationView interface can be obtained using the view() method on the IPresentationPlayer interface.

playbackController() → (non-null) {ispring.presenter.player.IPresentationPlaybackController}

Returns the IPresentationPlaybackController interface providing unrestricted presentation playback control facilities.

The unrestricted playback controller ignores the presentation playback and navigation restrictions (such as branching rules).

To obtain the IRestrictedPresentationPlaybackController interface which takes such restrictions into account, use the restrictedPlaybackController() method.

restrictedPlaybackController() → (non-null) {ispring.presenter.player.IRestrictedPresentationPlaybackController}

Returns the IRestrictedPresentationPlaybackController interface providing restricted playback control facilities.

The restricted playback controller takes the presentation playback and navigation restrictions (such as branching rules) into consideration.

To obtain the IPresentationPlaybackController interface which ignores such restrictions, use the playbackController() method.
