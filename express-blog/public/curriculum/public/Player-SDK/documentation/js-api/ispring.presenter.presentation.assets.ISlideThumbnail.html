load()

Loads the asset asynchronously if it hasn't been loaded.

Use the loadCompleteEvent() method to get the IEventDispatcher interface allowing to add or remove the loadComplete event handler.

Inherited From:
  • ispring.utils.assets.IAsset#load

loadCompleteEvent() → (non-null) {ispring.events.IEventDispatcher}

Returns the IEventDispatcher interface dispatching the loadComplete event when the asset has been loaded.

The returned event dispatcher passes the asset reference to the listener function. See the following example:

    var asset;
    // ...
    asset.loadCompleteEvent().addHandler(function(theAsset){
        window.console.log("The asset has been loaded successfully");
    });

    asset.load();
