interface
ispring.presenter.player.IPresentationPlaybackController
The IPresentationPlaybackController interface provides presentation playback and navigation control facilities.

To obtain the IPresentationPlaybackController interface use the playbackController() method on the IPresentationView interface.

nextSlideIndex() → {number}

Returns the next slide index taking hidden slides, custom slide shows, and slide branching settings into account.

-1 is returned if the next slide is not available, for example, if the current slide is the last slide of the presentation, or if the current slide is not defined (the return value of the currentSlideIndex() method is -1).

playbackCompleteEvent() → (non-null) {ispring.events.EventDispatcher}

Returns the IEventDispatcher interface dispatching the playbackComplete event when the presentation playback is complete.

The returned event dispatcher invokes the listener function without any parameters. See the following example:

    var presentationController;
    // ...
    presentationController.playbackCompleteEvent().addHandler(function(){
        window.console.log("Presentation playback has been completed.");
    });
Returns:
The IEventDispatcher interface dispatching the playbackComplete event when the presentation playback is complete.
Type: ispring.events.EventDispatcher

slideChangeEvent() → (non-null) {ispring.events.IEventDispatcher}

Returns the IEventDispatcher interface dispatching the slideChange event when the current slide changes.

The returned event dispatcher passes a new slide index to the listener function. See the following example.

    var presentationController;
    // ...
    presentationController.slideChangeEvent().addHandler(function(slideIndex){
        window.console.log("Current slide has been changed: " + slideIndex);
