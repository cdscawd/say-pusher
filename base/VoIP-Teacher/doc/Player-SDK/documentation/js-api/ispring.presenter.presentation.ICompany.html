interface
ispring.presenter.presentation.ICompany
The ICompany interface provides the company information associated with the presentation or a presenter.

To obtain company information of the loaded presentation use the company() method on the IPresentation interface. To obtain company information of a particular presenter use the company() method on the IPresenter interface.
