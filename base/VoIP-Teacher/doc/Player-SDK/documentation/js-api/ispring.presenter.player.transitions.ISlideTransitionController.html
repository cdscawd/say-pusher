interface
ispring.presenter.player.transitions.ISlideTransitionController
The ISlideTransitionController interface provides information about the current state and progress of the slide transition effect.

To obtain the ISlideTransitionController interface use the slideTransitionController() method on the IPresentationPlaybackController interface.

See also
  • ispring.presenter.player.IPresentationPlaybackController#slideTransitionController

transitionEffectCompleteEvent() → (non-null) {ispring.events.IEventDispatcher}

Returns the IEventDispatcher interface dispatching the transitionEffectComplete event when a slide transition effect is started.

The event dispatcher passes a new slide index to the event handling function. See the following example:

    var slideTransitionController;
    // ...
    slideTransitionController.transitionEffectCompleteEvent().addHandler(function(slideIndex){
        window.console.log("Transition to slide #" + slideIndex + " has been completed.");
    });
Returns:
Type: ispring.events.IEventDispatcher
