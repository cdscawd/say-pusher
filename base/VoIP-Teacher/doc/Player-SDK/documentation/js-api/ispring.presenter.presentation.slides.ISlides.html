interface
ispring.presenter.presentation.slides.ISlides
The ISlides interface represents a collection of the presentation slides.

To obtain the ISlides interface use the slides() method on the IPresentation interface.

See also
  • ispring.presenter.presentation.IPresentation#slides
  • ispring.presenter.presentation.slides.ISlide

convertTimestampToTime(timestamp non-null, considerTransitions opt, skipHiddenSlides opt) → {number}

Calculates the time offset from the beginning of the presentation to the specified presentation position.
Parameters:
Name Type Attributes Description
timestamp ispring.presenter.presentation.ITimestamp The position within the presentation represented by the ITimestamp interface to convert into the time offset.

You may pass an existing timestamp object obtained from the presentation player or create a new compatible timestamp using the createTimestamp() method.
considerTransitions boolean <optional>
 Indicates whether to include the slide transition duration when calculating the time offset, true by default.
skipHiddenSlides boolean <optional>
 Indicates whether to skip the presentation hidden slides when calculating the time offset, true by default.
See also
  • ispring.presenter.presentation.slides.ISlides#convertTimeToTimestamp
  • ispring.presenter.presentation.slides.ISlides#createTimestamp
Returns:
The time offset, in seconds, to the specified presentation position from the beginning of the presentation.
Type: number

duration() → {number}

Returns the total duration of the presentation slides, in seconds. The slide duration is the sum of the total duration of the animation steps and the transition effects duration.

To obtain the total duration of visible slides including the duration of their transition effects, use the visibleSlidesDuration() method.

To obtain the total duration of visible slides without the duration of their transition effects, use the visibleAnimationStepsDuration() method.

See also
  • ispring.presenter.presentation.slides.ISlides#visibleSlidesDuration
  • ispring.presenter.presentation.slides.ISlides#visibleAnimationStepsDuration
